# Machine Learning & Data Science

A collection of projects and notebooks documenting my work in machine learning and data science with Python. The focus is on understanding methods from the ground up — implementing core algorithms from scratch before using higher-level libraries — and applying them to real datasets.

## Focus areas

- Supervised and unsupervised learning
- Data preprocessing and feature engineering
- Model training, evaluation, and tuning
- Exploratory data analysis and visualization

## Projects

| Project | Description | Status |
|---|---|---|
| 01-linear-regression | Linear regression implemented from scratch (OLS, residual analysis, evaluation metrics) | In progress |
| 02-polynomial-regression | Polynomial regression from scratch | Planned |
| 03-logistic-regression | Logistic regression from scratch | Planned |
| 04-neural-networks | From intuition and manual implementation to TensorFlow | Planned |
| 05-unsupervised-learning | Clustering and dimensionality reduction | Planned |

## Project structure

Each project contains a Jupyter notebook or script, the dataset (or a link to it), and brief notes on the approach, key results, and limitations.
